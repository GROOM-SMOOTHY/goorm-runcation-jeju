# 폰트 가이드

이 페이지는 프로젝트에서 사용하는 폰트 설정을 정리한 문서입니다.  
실제 선언은 `src/styles/font.css` 에서 `@font-face`로 관리되고 있습니다.

<br/>
## 전체 기본 폰트

- **기본 폰트**: `Pretendard`, sans-serif
- **로고 폰트(SMOOTHY)**: `font-family: Plus Jakarta Sans`

export const familyClassMap = {
  Pretendard: "font-pretendard",
  Jua: "font-jua",
  "Plus Jakarta Sans": "font-plus-jakarta-sans",
};

export const FontBlock = ({ family, weight, description }) => {
  const familyClass = familyClassMap[family] ?? "";
  return (
    <div className='font-block'>
      <div className="font-block-header">
        <span className={`font-block-family ${familyClass}`} style={{ fontWeight: weight }}>{family}</span>
        <span className="font-block-weight">weight: {weight}</span>
      </div>
      <div className="font-block-description">{description}</div>
    </div>
  );
};

export const SampleText = ({ family, weight, label }) => {
  const familyClass = familyClassMap[family] ?? "";
  return (
    <div
      className={`sample-text ${familyClass}`}
      style={{ fontWeight: Number(weight) }}
    >
      <div className="sample-text-label">{label}</div>
      <div>제주 런케이션를 떠나는 가장 쉬운 방법을 소개합니다.</div>
      <div>Goorm Runcation Jeju – Work, Run, and Relax.</div>
    </div>
  );
};
