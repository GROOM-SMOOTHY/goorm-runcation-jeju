## 📝작업 내용

### SettleCard 컴포넌트 구현

정산 내역을 표시하는 카드 컴포넌트를 구현했습니다. 사용자가 정산 정보를 확인하고 자신의 입금 상태를 관리할 수 있습니다.

#### 주요 기능

1. **접기/펼치기 기능**
   - 헤더 클릭으로 카드 내용 접기/펼치기
   - Grid 애니메이션을 활용한 부드러운 전환 효과
   - 펼쳐진 상태에서 테두리 색상 변경으로 시각적 피드백 제공

2. **입금 상태 토글**
   - Radix UI Switch를 사용한 입금 완료/미완료 상태 전환
   - 토글 상태에 따라 "나"를 미완료/완료 목록에 동적으로 이동
   - 상태에 따른 아이콘 및 색상 변경 (완료: 초록색, 미완료: 주황색)

3. **정산 정보 표시**
   - 총 금액 (TOTAL AMOUNT)
   - 정산 진행률 (프로그래스바 + 완료 인원 수)
   - 1인당 금액 자동 계산 및 표시
   - 진행률에 따른 헤더 뱃지 자동 변경 ("정산완료" / "정산 미완료")

4. **계좌 정보 관리**
   - 받는 사람 계좌 정보 표시 (은행명, 마스킹된 계좌번호)
   - 계좌번호 복사 기능 (Clipboard API 사용)
   - 현재 사용자가 입금자일 경우 "나" 뱃지 표시

5. **멤버 목록 표시**
   - 미완료/완료 멤버를 3열 그리드로 표시
   - 각 멤버의 아바타 및 이름 표시
   - 현재 사용자("나")는 항상 맨 앞에 정렬
   - 현재 사용자에게는 "나" 뱃지 및 테두리 강조 표시
   - 이름이 길 경우 말줄임 처리 (3글자까지 표시)

#### 기술 스택

- **UI 라이브러리**: Radix UI (`@radix-ui/react-switch`, `@radix-ui/react-progress`)
- **아이콘**: React Icons (`react-icons/fa`, `react-icons/md`)
- **스타일링**: CSS Modules
- **상태 관리**: React Hooks (`useState`)
- **토스트 알림**: 공통 Toast 컴포넌트 사용

#### 컴포넌트 구조

```
SettleCard
├── Header (접기/펼치기 버튼)
│   ├── 아이콘 블록
│   ├── 제목 및 날짜
│   └── 상태 뱃지 (정산완료/정산 미완료)
└── Body (펼쳐질 때 표시)
    ├── 입금 상태 토글
    ├── 총 금액 섹션
    ├── 정산 진행률 섹션 (공통 Progress 컴포넌트)
    ├── 1인당 금액
    ├── 계좌 정보 섹션
    ├── 미완료 멤버 섹션
    └── 완료 멤버 섹션
```

#### 주요 로직

- **동적 멤버 목록 관리**: 토글 상태에 따라 현재 사용자를 미완료/완료 목록에 자동으로 이동
- **진행률 계산**: 완료 인원 수를 기반으로 진행률 자동 계산 및 프로그래스바 업데이트
- **정렬 로직**: 현재 사용자를 항상 목록의 맨 앞에 배치하여 가독성 향상

#### 접근성 (A11y)

- `aria-expanded`, `aria-label`, `aria-hidden` 속성 사용
- 키보드 접근성 고려 (button 요소 사용)
- 스크린 리더를 위한 적절한 라벨 제공
